# 🎮 Game Hub

Game Hub is a web-based mini gaming platform where users can log in and play interactive browser games like **Simon Game** and **Drum Kit**.

---

## 🚀 Features

- 🔐 Login system using localStorage
- 🎮 Play mini games (Simon, Drum Kit)
- 🎨 Neon-themed modern UI
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 📂 Project Structure

Game-Hub/

│── hub.js

│── images/  

---

## ▶️ How to Run

1. Clone repo
2. Open `index.html` in browser
